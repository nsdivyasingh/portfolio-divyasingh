## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

This project is configured for Vercel deployment. Simply push to your repository and Vercel will automatically deploy.

## Design Philosophy

The portfolio follows a "Dark Mode Glam + Constellation Universe" aesthetic:
- Deep space backgrounds (near-black with subtle gradients)
- Small glowing stars scattered across the site
- Constellation lines appearing subtly on scroll and hover
- Projects represented as glowing planets
- Elegant, mysterious, confident tone
